# mutsu — 2026年5月の実装状況

## 概要

5月は **実用性重視への方針転換** を行った月。roast テスト通過率の追求から、実際にユーザーが使えるインタープリタとしての品質向上にシフトした。

- Whitelist: 1103 → 1148 (+45)
- PR マージ: 45件以上
- PLAN.md 策定・Phase 1〜6 をほぼ完了

## ハイライト

### デバッガビリティの大幅改善 (Phase 1)

エラーメッセージが劇的に改善された。

**Before:**
```
Runtime error: oops
```

**After:**
```
oops
  in sub bar at script.raku line 3
  in sub foo at script.raku line 2
  in block <unit> at script.raku line 4
```

- ランタイムエラーに行番号・ファイル名・バックトレース表示 (#2203)
- `$!.line`, `$!.file`, `$!.backtrace` メソッド (#2206)
- `.backtrace` が Backtrace オブジェクトを返す (frames, .list, .Str) (#2215)
- パースエラーに `===SORRY!===` 形式のコードスニペット表示 (#2218)
- メソッド名 typo 時に「Did you mean ...?」サジェスション (#2211)

### Grammar / Match の実用化 (Phase 2)

- `unit grammar` サポート (#2220)
- regex alias (`<name=.subrule>`) (#2220)
- proto token `<sym>` (#2220)
